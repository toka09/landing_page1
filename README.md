# Portfolio Website

This is a modern, responsive portfolio website built with html and  CSS. It features sections for About, Projects, and Contact, with clean design and smooth hover effects.

---

## Demo

To see the live version, run the project locally or deploy it to your preferred hosting platform.

## Features

- Responsive layout for desktop and mobile
- Smooth hover animations
- Customizable sections for your content
- Easy to update images, text, and links
